# j2y
json to yaml tool。

# 使用
```shell
./j2y -h
Error: required flag(s) "input" not set
Usage:
  json2yaml [flags]

Flags:
  -h, --help            help for json2yaml
  -i, --input string    输入 JSON 文件
  -o, --output string   输出 YAML 文件

required flag(s) "input" not set
```
可以不设置输出文件。如果不设置，默认为标准输出。
